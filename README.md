# ⚖️ MassConvert — Conversor de Peso Profesional

**MassConvert** es una aplicación web moderna, rápida e intuitiva diseñada para realizar conversiones de unidades de masa y peso en tiempo real con precisión. Construida desde cero usando tecnologías web nativas, cuenta con una interfaz con estética **Neumórfica** y soporte para accesibilidad.

## ✨ Características Principales

- **Conversión Multidireccional en Tiempo Real:** Intercambia entre Kilogramos ($kg$), Libras ($lbs$), Gramos ($g$) y Onzas ($oz$).
- **Interfaz Neumórfica Modernizada:** Diseño visual táctil y limpio basado en sombras suaves y paleta de colores personalizada.
- **Copiado al Portapapeles con un Clic:** Botón integrado con respuesta visual instantánea.
- **Validación de Datos en Vivo:** Manejo robusto de errores para prevenir entradas no válidas.
- **Atajos de Teclado:** Presiona `Escape` para resetear los valores y volver al input al instante.
- **Código Válido y Accesible:** Estructura que cumple con los estándares oficiales de la **W3C** e incluye etiquetas accesibles para lectores de pantalla.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica completa.
- **CSS3:** Custom Properties (Variables CSS), Flexbox, transiciones suaves y arquitectura Neumórfica.
- **JavaScript (ES6+):** Lógica modular, manipulación del DOM y API de Clipboard (`navigator.clipboard`).
